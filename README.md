# Phelab Homebrew Tap

This repository hosts the Homebrew formulas for [Phelab CLIs](https://github.com/phe-lab).

## Install

```bash
brew tap phe-lab/tap
brew install <tool>
```

Available versions:
- `ws` (latest version)

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
